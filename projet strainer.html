3D Modeling – Piping System using SolidWorks

Project Description

Creation of a 3D model of a fluid transfer piping system using SolidWorks.

The system consists of the following components:

  • One pump
  • Two tanks
  • One strainer filter
  • Piping network

Project Details

The project focused on building the full 3D assembly using the Routing module to define the pipe paths and integrate standard components such as elbows, flanges, and fittings.

The objective was to accurately represent the fluid flow circuit from the tank, through the pump, and toward the strainer, ensuring proper geometric alignment and mechanical assembly consistency.
